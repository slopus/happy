# Task 002: 会话列表智能排序

## 目标

重新组织 Happy 项目的会话（sessions）列表显示顺序，按照以下优先级排序：

### 排序维度（优先级从高到低）

1. **是否需要人介入操作** - 有权限请求（permission requests）
2. **是否需要人输入命令** - Claude 等待用户输入（waiting for input）
3. **是否已完成** - 任务已完成（completed）
4. **最近更新时间** - 最近活动的会话优先

### 会话状态类型

根据代码分析，会话（Session）可能有以下状态：

1. **权限请求状态**：
   - `session.agentState.requests` - 待处理的权限请求

2. **等待输入状态**：
   - `session.agentState.state === 'ready'` - 等待用户输入
   - `session.agentState.state === 'waiting'` - 等待响应

3. **工作状态**：
   - `session.agentState.state === 'thinking'` - AI 正在思考
   - `session.agentState.state === 'executing'` - 正在执行命令

4. **完成/离线状态**：
   - `session.presence === 'offline'` - 会话离线
   - 无活动状态

## 排序算法设计

### 优先级规则

```typescript
// 优先级权重（数字越大优先级越高）
const PRIORITY_WEIGHTS = {
    HAS_PERMISSION_REQUEST: 1000,  // 最高优先级：需要人介入
    WAITING_FOR_INPUT: 500,        // 高优先级：需要人输入命令
    ACTIVE_THINKING: 100,          // 中优先级：正在工作
    COMPLETED_OFFLINE: 0,          // 低优先级：已完成/离线
};
```

### 排序逻辑

```typescript
function getSessionPriority(session: Session): number {
    // 1. 检查是否有权限请求
    if (session.agentState?.requests && Object.keys(session.agentState.requests).length > 0) {
        return PRIORITY_WEIGHTS.HAS_PERMISSION_REQUEST;
    }

    // 2. 检查是否等待输入
    if (session.agentState?.state === 'ready' || session.agentState?.state === 'waiting') {
        return PRIORITY_WEIGHTS.WAITING_FOR_INPUT;
    }

    // 3. 检查是否正在工作
    if (session.agentState?.state === 'thinking' || session.agentState?.state === 'executing') {
        return PRIORITY_WEIGHTS.ACTIVE_THINKING;
    }

    // 4. 其他情况（离线/完成）
    return PRIORITY_WEIGHTS.COMPLETED_OFFLINE;
}

function sortSessions(sessions: Session[]): Session[] {
    return sessions.sort((a, b) => {
        // 首先按优先级排序
        const priorityDiff = getSessionPriority(b) - getSessionPriority(a);
        if (priorityDiff !== 0) {
            return priorityDiff;
        }

        // 优先级相同时，按最近更新时间排序
        const timeA = a.metadata?.updatedAt || a.metadata?.createdAt || 0;
        const timeB = b.metadata?.updatedAt || b.metadata?.createdAt || 0;
        return timeB - timeA;
    });
}
```

## 数据结构假设

需要验证的 Session 数据结构：

```typescript
interface Session {
    id: string;
    presence: 'online' | 'offline';
    agentState?: {
        state: 'ready' | 'waiting' | 'thinking' | 'executing' | 'idle';
        requests?: Record<string, PermissionRequest>; // 权限请求
    };
    metadata?: {
        createdAt?: number;
        updatedAt?: number;
        path?: string;
        machineId?: string;
    };
}
```

## 边界情况处理

1. **会话无 agentState**：视为低优先级（已完成）
2. **时间戳缺失**：使用创建时间或当前时间
3. **多个权限请求**：统一按"需要人介入"处理
4. **离线但有未完成任务**：需要特殊标记

describe('sessionSort', () => {
    it('should prioritize sessions with permission requests', () => {
        const sessions = [
            { id: 'a', agentState: { state: 'ready' } },
            { id: 'b', agentState: { state: 'thinking', requests: { req1: {} } } },
        ];
        const sorted = sortSessionsByPriority(sessions);
        expect(sorted[0].id).toBe('b'); // 有权限请求的优先
    });

    it('should prioritize waiting sessions over active ones', () => {
        const sessions = [
            { id: 'a', agentState: { state: 'thinking' } },
            { id: 'b', agentState: { state: 'ready' } },
        ];
        const sorted = sortSessionsByPriority(sessions);
        expect(sorted[0].id).toBe('b'); // 等待输入优先于工作中
    });

    it('should sort by time when priorities are equal', () => {
        const sessions = [
            { id: 'a', agentState: { state: 'ready' }, metadata: { updatedAt: 100 } },
            { id: 'b', agentState: { state: 'ready' }, metadata: { updatedAt: 200 } },
        ];
        const sorted = sortSessionsByPriority(sessions);
        expect(sorted[0].id).toBe('b'); // 更新时间更晚的优先
    });
});
