# Task 001: 独立语音输入模式

## 目标
实现一个独立的语音输入功能，与现有的实时对话（ElevenLabs）分离：
- 右下角添加文字/语音模式切换按钮
- 语音模式下显示大录音按钮（中间）
- 点击录音 → ASR 识别 → 填充到输入框 → 用户手动发送

## 架构设计

### 当前实现分析

#### AgentInput 组件 (`sources/components/AgentInput.tsx`)
**现有功能：**
- 右下角是"发送/麦克风"按钮（Line 952-1016）
  - 有文字时显示"发送"按钮（箭头向上图标）
  - 无文字且有 `onMicPress` 时显示麦克风按钮
  - 当前麦克风按钮触发的是实时语音对话（ElevenLabs）

**修改点：**
1. 将右下角按钮改为**文字/语音模式切换**按钮
2. 发送按钮移到输入框内部右侧（或左下角）
3. 语音模式下，中间显示大录音按钮

#### SessionView 组件 (`sources/-session/SessionView.tsx`)
**现有功能：**
- Line 212-233: `handleMicrophonePress` 处理麦克风按钮，启动/停止 ElevenLabs 实时会话
- Line 300-301: 将 `micButtonState` 传递给 `AgentInput`

**修改点：**
1. 添加新的"录音"处理函数（ASR 模式）
2. 保留原有 `handleMicrophonePress`（以后可能还需要实时对话功能）
