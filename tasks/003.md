# Task 003: 支持复制用户自己的发言

## 目标

允许用户复制自己发送的消息内容，方便重用、编辑或分享。

## 用户场景

1. **重用命令**：用户想复制之前发送的命令，稍作修改后重新发送
2. **分享内容**：用户想将自己的问题复制到其他地方
3. **记录保存**：用户想保存自己的某些提问作为记录
4. **编辑重发**：发现之前的问题表达不清，想复制后重新编辑

## 技术实现

### 1. 识别用户消息

```typescript
// 判断消息是否为用户发送
function isUserMessage(message: Message): boolean {
    // 根据实际数据结构判断
    return message.role === 'user' || message.type === 'user';
}
```

### 2. 复制文本到剪贴板

```typescript
import * as Clipboard from 'expo-clipboard';

async function copyMessageToClipboard(text: string) {
    try {
        await Clipboard.setStringAsync(text);
        // 显示成功提示
        showToast('已复制到剪贴板');
    } catch (error) {
        console.error('复制失败:', error);
        showToast('复制失败');
    }
}
```

### 3. 提取消息文本内容

```typescript
function getMessageText(message: Message): string {
    // 根据实际消息结构提取文本
    if (typeof message.content === 'string') {
        return message.content;
    }
    // 处理复杂格式（Markdown、多段落等）
    return extractTextFromContent(message.content);
}
```

### 4. 长按交互（移动端）

```typescript
<Pressable
    onLongPress={() => handleLongPress(message)}
    delayLongPress={500} // 500ms 长按触发
>
    <UserMessageContent message={message} />
</Pressable>
```

### 5. Action Sheet / Modal

使用项目现有的 Modal 组件：

```typescript
import { Modal } from '@/modal';

function showMessageActions(message: Message) {
    Modal.actionSheet([
        {
            text: t('message.copy'),
            icon: 'copy-outline',
            onPress: () => copyMessageToClipboard(getMessageText(message))
        },
        {
            text: t('message.resend'),
            icon: 'refresh-outline',
            onPress: () => resendMessage(message)
        },
        {
            text: t('common.cancel'),
            style: 'cancel'
        }
    ]);
}
```

## 边界情况处理

1. **空消息**：禁用复制功能或提示"无内容可复制"
2. **超长消息**：正常复制，不做截断
3. **格式化内容**：
   - Markdown：保留原始 Markdown
   - 代码块：保留代码格式
   - 多段落：保留换行
4. **特殊字符**：正确处理 emoji、中文等
5. **图片/附件**：仅复制文本部分

## 性能考虑

- 复制操作应异步执行，避免阻塞 UI
- 长文本复制应有加载指示
- 避免重复渲染复制按钮

## 安全考虑

- 不复制敏感信息（API keys、密码等）- 未来可添加检测
- 复制前检查剪贴板权限
- 处理复制失败的情况
