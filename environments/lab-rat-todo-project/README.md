# Lab Rat Todo Project

A tiny frontend-only todo app used as the default Happy integration fixture.

## Files

- `index.html` - app shell
- `styles.css` - layout and theme
- `app.js` - todo logic and local storage persistence

## Why it exists

- zero install
- realistic enough for end-to-end UI edits
- safe to copy into every fresh environment
